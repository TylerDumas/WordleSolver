# WordleSolver
>Solves the Wordle! puzzle game.

## Technologies
Project is created with:
* JavaScript
* React
* Electron.js


## Building and Running
Ensure that Node is installed and up to date.

Windows Node Installation:
```
 - curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash  # install Node Version Manager
 - nvm install node  # install Node (v25.2.1 as of writing this.)
 - source ~/.bashrc  # reload your shell
```

Run WordleSolver
```
 - git clone https://github.com/TylerDumas/WordleSolver.git
 - sudo apt install npm  # ensure npm is installed
 - cd WordleSolver/wordle-solver
 - npm install
 - npm run start
```
