# WordleSolver
>Solves the Wordle! puzzle game.

## Technologies
Project is created with:
* JavaScript
* React
* Electron.js


## Building and Running
```
 - git clone https://github.com/TylerDumas/WordleSolver.git
 - cd wordle-solver
 - npm install
 - npm run electron-dev
```
